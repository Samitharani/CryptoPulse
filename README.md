# CryptoPulse
Real-time cryptocurrency analysis and awareness platform



CryptoPulse is a web-based platform that provides real-time cryptocurrency price tracking, historical trend analysis, and educational market insights to help beginners understand crypto market behavior.

## Features
- Live cryptocurrency prices using CoinGecko API
- Interactive charts and market trends
- Educational insights for beginners
- Machine learning-based trend estimation (LSTM)

## Tech Stack
- HTML, CSS, JavaScript
- Python, Flask
- TensorFlow / Keras
- CoinGecko API
- Yahoo Finance API

## Usage
This project is intended for educational and analytical purposes only and does not provide financial or investment advice.
